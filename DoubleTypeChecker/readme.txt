DoubleTypeChecker(DTC) ver1.12a
2012/11/26

この度は本ソフトをダウンロードして頂きありがとうございます。


■このソフトは？
ポケモンの複合タイプの相性を考えるのが面倒だ、という人のための補助ソフトです。
パーティが苦手としている複合タイプを教えてくれたり、
新たに複合タイプを加えるならどれがいいのかをオススメしてくれます。


■動作環境
Java Runtime Environment Version 7 以上
Javaさえ入っていればOSを問わず動作しますが、
JRE6以下は恐らく動作しません。
動作しない場合はJRE6をアンインストールし、
http://java.com/ から再インストールしてください。


■使い方
・起動について
[Windows]DoubleTypeChecker.jarをダブルクリックすれば起動します。
[Mac/Linux]ダブルクリック起動は設定すれば可能ですが、
できない場合はターミナル/端末から
「java -jar DoubleTypeChecker.jar」で実行してください。


・データについて
「DTCsave.dat」というファイルを必要な場面で自動的に同じフォルダに生成します。
変にいじったりして正常に動作しない場合は思い切って「DTCsave.dat」を削除してください。

また、この複合は実在しない！第四世代以前を考えたい！という場合を取り除いたりしたい場合は
自分で探索範囲を編集することができます(My設定)。
ソフト起動時に「DTCtype.txt」が自動生成されるのでコメントにしたがって編集してください。
また変ないじり方をして正常に動作しない場合は同様に「DTCtype.txt」を削除してください。


・使用方法について
左側に自分のパーティや考えているパーティのタイプや特性を入力してください。
横のチェックの入ってる部分しか考慮しません。

複合タイプのデータはポケモンがポケモン（第五世代）のよくあるルール、
ポケ伝説がポケモン（第五世代）の伝説有り、
全部が全複合となります。
My設定については上述の通り。

候補指定にチェックが入ってるとメンバー候補を上記の環境データとは別に指定できます。
ただしポケモンの場合に限り、存在しないタイプは候補からも排除されます。

受け専にチェックが入ってると純粋に攻撃を受ける上でのタイプ相性を考えます。

カバーにチェックが入ってるとパーティの良い点で悪い点を相殺します。
入ってないと欠点しか見ません。

−基準は数が少なければ少ないほどチェックが厳しくなります。
＋基準は逆に数が大きければ大きいほどチェックが厳しくなります。

各機能についてはマウスポインタを乗せるとヘルプが出るのでそちらも参照ください。


・用途別の推奨設定（一例）
アタッカーだけで構成するからなるべく多くのタイプを倒したい
→-6 または +2

各メンバーのポテンシャルで押し切るのでとにかく相性を安定させたい
→カバーON、-1 または +6

もちろん安定させていたいけど弱点をつけないタイプがいるのはいやだ
→カバーON、-1 かつ +2

受け出しに不満がある
→受け専ON、カバーON、-3 または +6


■真ん中の表の相性リスト(例：{0,-1,2,})って何？
各メンバーとその複合タイプとの相性のスコアを表しています。
メンバーとリストの順番は対応しています。


■スコアの算出方法について
より相手のタイプに有効な方のタイプを使って攻撃したとき、
無効が-3点、半減が-1点、等倍が0点、弱点が1点、重複弱点が2点と設定してます。
このようにして出される「敵の複合タイプ→自分の複合タイプ」を引いた点数がスコアになります。
さらに受け専で無い場合は「自分の複合タイプ→敵の複合タイプ」がスコアに加点されます。

つまり（基本的に）タイプ一致だけで適切に殴り合って、どちらに分があるかが算出されます。
またサブウェポン・素早さは全く考慮していません。


■盲点基準・盲点数と得意基準・得意数について
・盲点関連（−）
相性リストの負の部分の数を数えて（カバーがONの場合そこから正の数を引く）、
その数が盲点基準を超えると対処できないと判断されます。

・得意関連（＋）
相性リストの正の部分の数を数えて、
その数が得意基準を超えると対処できると判断されます。

つまり次の式を満たすときに苦手と判断されます。
（盲点数≧盲点基準）かつ/または（選んだ方と逆）（得意数＜得意基準）


■あのファイルを読み込まなくなって困った
とりあえずワケが分からなくなったら消しましょう。
必要なファイルが無い状態は自動生成しなおします。
