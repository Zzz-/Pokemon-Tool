DoubleTypeChecker(DTC) ver1.02
2012/11/17

この度は本ソフトをダウンロードして頂きありがとうございます。


■このソフトは？
ポケモンの複合タイプの相性を考えるのが面倒だ、という人のための補助ソフトです。
パーティが苦手としている複合タイプを教えてくれたり、
新たに複合タイプを加えるならどれがいいのかをオススメしてくれます。


■動作環境
Java Runtime Environment Version 7 以上
Javaさえ入っていればOSを問わず動作しますが、
JRE6以下は恐らく動作しません。
動作しない場合はJRE6をアンインストールし、
http://java.com/ から再インストールしてください。


■使い方
・起動について
[Windows]DoubleTypeChecker.jarをダブルクリックすれば起動します。
[Mac/Linux]ダブルクリック起動は設定すれば可能ですが、
できない場合はターミナル/端末から
「java -jar DoubleTypeChecker.jar」で実行してください。


・データについて
「DTCsave.dat」というファイルを必要な場面で自動的に同じフォルダに生成します。
変にいじったりして正常に動作しない場合は思い切って「DTCsave.dat」を削除してください。


・使用方法について
左側に自分のパーティや考えているパーティのタイプを入力してください。
タイプの横のチェックの入ってる部分しか考慮しません。

ポケモンにチェックが入ってるとポケモンで存在するタイプのみ（BW2現在）を考えます。
更に エスパー/ドラゴン→浮遊 のようにそのタイプはその特性しかいない、という場合は
その特性であるという前提で相性を算出します（敵のみ）。

ポケモンと伝説にチェックが入ってると伝説でしか存在しないタイプも考慮します。
（準伝説は伝説扱いではありません）

受け専にチェックが入ってると純粋に攻撃を受ける上でのタイプ相性を考えます。

カバーにチェックが入ってるとパーティの良い点で悪い点を相殺します。
入ってないと欠点しか見ません。

盲点基準は数が少なければ少ないほどチェックが厳しくなります。オススメは２〜３です。

各機能についてはマウスポインタを乗せるとヘルプが出るのでそちらも参照ください。


■スコアの算出方法について
より相手のタイプに有効な方のタイプを使って攻撃したとき、
無効が-3点、半減が-1点、等倍が0点、弱点が1点、重複弱点が2点と設定してます。
このようにして出される「敵の複合タイプ→自分の複合タイプ」を引いた点数がスコアになります。
さらに受け専で無い場合は「自分の複合タイプ→敵の複合タイプ」がスコアに加点されます。

つまり（基本的に）タイプ一致だけで適切に殴り合って、どちらに分があるかが算出されます。
またサブウェポン・素早さは全く考慮していません。
