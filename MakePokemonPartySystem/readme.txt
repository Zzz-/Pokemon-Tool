MakePokemonPartySystem(MPPS) ver1.03
2012/8/2

この度は本ソフトをダウンロードして頂きありがとうございます。

■このソフトは？
ポケモンをたくさん育ててしまったけど選択肢が多すぎて
どのポケモンを使えばいいのか分からない、といった
熱心なブリーダーのためのパーティ自動生成ツールです。
それなりに強いパーティを構築する事も可能です。

■動作環境
Java Runtime Environment Version 7 以上
Javaさえ入っていればOSを問わず動作しますが、
JRE6以下は恐らく動作しません。
動作しない場合はJRE6をアンインストールし、
http://java.com/ から再インストールしてください。

■使い方
・起動について
[Windows]MPPS.jarをダブルクリックすれば起動します。
[Mac/Linux]ダブルクリック起動は設定すれば可能ですが、
基本的にターミナル/端末から「java -jar MPPS.jar」で実行してください。

・データについて
datフォルダ内のファイルを読み書きします。
フォルダやファイルが無い場合は自動生成されます。

・使用方法について
当ソフトではポケモンをグループで管理しています。
どのデータのどのグループを使用するかを選択し、生成を押すと
該当データからパーティを自動生成します。
各機能についてはマウスポインタを乗せるとヘルプが出るのでそちらを参照ください。
